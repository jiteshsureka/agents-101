THE PRINCIPLEনীতি

Progressive Disclosureপ্রগ্রেসিভ ডিসক্লোজার

Load the least possible up front.
Reveal more only when needed. শুরুতে যত কম সম্ভব লোড করো।
প্রয়োজনে আরও দেখাও।
This is a UX principle borrowed for AI skill design: don't show the whole manual the moment someone walks in the door. Give them the handle; they'll ask for the manual when they need it.এটি UI/UX থেকে ধার করা নীতি, এখন AI স্কিল ডিজাইনে প্রয়োগ: কেউ দরজায় ঢোকামাত্র পুরো ম্যানুয়াল দেখিয়ো না। হ্যান্ডেল দাও; প্রয়োজনে তারা ম্যানুয়াল চাইবে।

TIER 3 — PART Aস্তর ৩ — অংশ A

Bundled Scripts — Keep Logic Out of the Bodyবান্ডেলড স্ক্রিপ্ট — লজিক বডিতে রাখো না

A skill can ship a Python or Bash script alongside its SKILL.md. Instead of inlining 150 lines of validation logic, the body simply says: "run scripts/validate.py".একটি স্কিল তার SKILL.md-এর পাশে Python বা Bash স্ক্রিপ্ট পাঠাতে পারে। ১৫০ লাইনের ভ্যালিডেশন লজিক ইনলাইন করার বদলে বডি শুধু বলে: "scripts/validate.py চালাও"।
AGENT "I see the SKILL.md says: run scripts/run.py with the collected params. Let me execute that now."
← The script body never sits in the context window unless the agent reads the file. "দেখছি SKILL.md বলছে: কালেক্ট করা প্যারাম দিয়ে scripts/run.py চালাও। এখন এটি চালাই।"
← স্ক্রিপ্টের বডি কনটেক্সট উইন্ডোতে থাকে না যতক্ষণ না এজেন্ট ফাইলটি পড়ে।
scripts/validate.pyscripts/validate.py scripts/run.pyscripts/run.py scripts/generate.shscripts/generate.sh
The script executes when needed — the body only carries the instruction to run it. No wasted tokens on code that sits dormant.স্ক্রিপ্ট দরকার হলে চলে — বডিতে শুধু সেটি চালানোর নির্দেশ থাকে। নিষ্ক্রিয় কোডে কোনো টোকেন নষ্ট হয় না।
TIER 3 — PART Bস্তর ৩ — অংশ B

Reference Files — Move Heavy Docs Outরেফারেন্স ফাইল — ভারী ডকস বাইরে সরাও

Large checklists, API field tables, big templates — move them into references/. SKILL.md just points to them with a single line.বড় চেকলিস্ট, API ফিল্ড টেবিল, বড় টেমপ্লেট — references/-এ সরাও। SKILL.md শুধু একটি লাইনে তাদের দিকে নির্দেশ করে।
# In SKILL.md — pointing to a reference, NOT inlining it ## When you need the full API field list: Read references/api-fields.md ## When you need the release checklist: Read references/release-checklist.md # The files are NOT in the context window until the agent reads them.

FOLDER STRUCTUREফোল্ডার কাঠামো

The Three-Tier Layoutতিন-স্তরের কাঠামো

WATCH OUTসাবধান

Four Gotchas to Avoidচারটি ভুল এড়াও

INLINE EVERYTHINGসব ইনলাইন করা
The habit that created the 800-line problem in the first place. Every "it's easier to paste it here" becomes permanent token tax. যে অভ্যাসটি শুরুতে ৮০০ লাইনের সমস্যা তৈরি করেছিল। প্রতিটি "এখানে পেস্ট করা সহজ" স্থায়ী টোকেন কর হয়ে যায়।
FORGET THE POINTERপয়েন্টার ভুলে যাওয়া
Moving a file to references/ without updating SKILL.md to mention it. The agent never knows to read it. SKILL.md আপডেট না করে ফাইল references/-এ সরানো। এজেন্ট কখনো জানে না সেটি পড়তে হবে।
UNDOCUMENTED SCRIPTSঅডকুমেন্টেড স্ক্রিপ্ট
Shipping scripts/run.py with no usage note in SKILL.md. The agent doesn't know what args it takes or when to call it. SKILL.md-এ usage নোট ছাড়া scripts/run.py পাঠানো। এজেন্ট জানে না কী আর্গুমেন্ট লাগে বা কখন ডাকতে হয়।
