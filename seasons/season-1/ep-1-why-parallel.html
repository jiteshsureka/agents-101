GREEN LIGHT

Tasks That Run Well in Parallel

  • ✓ Independent features — no shared code between them
  • ✓ Tests for code you finished yesterday
  • ✓ API docs, changelogs, or release notes
  • ✓ Work across separate microservices or modules
  • ✓ Isolated refactors with clean module boundaries
The test: if two tasks don't touch the same files and don't need each other's output — they can run in parallel.
RED LIGHT

Tasks That Don't Parallelize

  • ✗ Task B needs Task A's output before it can start
  • ✗ Both sessions would modify the same source file
  • ✗ Session B depends on Session A's running service
  • ✗ Deeply coupled changes — one PR breaks the other
The parallel check: "Could I hand these two tasks to different people right now?" If no — don't parallelize yet.

WATCH OUT

Parallel Isn't Free

  • ⚠️
    Cost: 3 sessions ≈ 3× token usage. Focused prompts reduce waste.
  • ⚠️
    Context tax: Each session needs its own full context from scratch. Write it clearly.
  • ⚠️
    Merge overhead: More PRs = more review cycles. Smaller PRs help compensate.
  • ⚠️
    Mental load: Context-switching sessions has a real cognitive cost.
WHEN IT PAYS

The Parallelism Payoff Test

WORTH IT WHEN…
  • ✓ Each task takes > 20 minutes
  • ✓ Tasks are fully independent
  • ✓ Merge is straightforward
  • ✓ You have 2+ tickets queued
  • ✓ Sessions don't share any files
SKIP IT WHEN…
  • ✗ Tasks are short (< 15 min)
  • ✗ Deep dependency between tickets
  • ✗ Only 1 ticket in the queue
  • ✗ Already in deep flow on one task
  • ✗ Touching the same config/schema
Default trigger: start a new session whenever you notice yourself waiting on Claude. That moment is your signal.
REAL TALK

Context-Switching Has a Cost

HABIB · BACKEND ENGINEER "First week I tried this, I kept forgetting which session I was in. I pasted the wrong fix into the wrong terminal twice."
Fix: set a 20–25 minute rotation timer. Before you switch sessions, write one line in a scratch note: where you left off and what the session was doing. Treat it like a standup for your future self.
