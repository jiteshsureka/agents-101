THE SHIFT

From assistant to team. সহকারী থেকে দলে।

Beforeআগে

One Claude. One context. One stream of work. Hit a huge task → run out of context → restart.একটা Claude। একটা context। একটা কাজের ধারা।

Swarmsঝাঁক

Lead agent decomposes the task → spawns specialist agents → they run in parallel, message each other → converge on one result.লিড এজেন্ট কাজ ভাগ করে → বিশেষজ্ঞ এজেন্ট স্পন করে → সমান্তরালে চলে → একটা ফলাফলে মেলে।

The key insight: LLMs reason worse as context grows. Keep each agent small, focused, and fast. মূল উপলব্ধি: context বাড়লে LLM দুর্বল হয়। প্রতিটা এজেন্টকে ছোট, কেন্দ্রীভূত রাখো।

AGENT TEAMS · TASK BOARD

Shared task board — no collisions. শেয়ার্ড টাস্ক বোর্ড — সংঘর্ষ নেই।

  • 🔒 File locking — two agents can't edit the same file simultaneously. ফাইল লকিং — দুই এজেন্ট একই ফাইলে একসাথে কাজ করতে পারে না।
  • 🔗 Dependencies — task B auto-unblocks when task A completes. ডিপেন্ডেন্সি — টাস্ক A শেষ হলে টাস্ক B স্বয়ংক্রিয়ভাবে শুরু হয়।
  • 📊 Plan approval gates — lead presents plan before risky work begins. প্ল্যান অ্যাপ্রোভাল — ঝুঁকিপূর্ণ কাজের আগে অনুমোদন নেওয়া হয়।
  • 🎯 Delegate mode — restrict lead to coordination only. Teammates do all writing. ডেলিগেট মোড — লিড শুধু সমন্বয় করে। টিমমেটরা সব লেখে।

Result: no race conditions, no overwritten work, clear dependency chains. ফলাফল: race condition নেই, ওভাররাইট নেই, পরিষ্কার dependency চেইন।

DYNAMIC WORKFLOWS · SETUP

Headless mode — the engine. হেডলেস মোড — ইঞ্জিন। 

# Run Claude non-interactively — orchestrator or single agent
claude -p "Migrate all fetch() calls in src/ from v1 to v2 API"

# Cap turns to prevent runaway cost
claude -p "Refactor auth logic in auth/service.ts" --max-turns 20

# Shell-level parallelism — spawn 10 at once, rate-limited
ls src/services/*.ts \
  | xargs -P 10 -I {} \
    claude -p "Migrate fetch() calls in {}"

-p = non-interactive (print mode). -P 10 = 10 concurrent shells. --max-turns 20 = hard stop. -p = নন-ইন্টারেক্টিভ। -P 10 = ১০টা একসাথে। --max-turns 20 = কঠোর সীমা।

DYNAMIC WORKFLOWS · PROMPTS

Scoped prompts. No shared memory. সীমাবদ্ধ প্রম্পট। শেয়ার্ড মেমরি নেই।

Vague — burns tokensঅস্পষ্ট — টোকেন নষ্ট

"Migrate all API calls."

Agent doesn't know which file, which version, which output format. Wasted run.এজেন্ট জানে না কোন ফাইল, কোন ভার্সন। ব্যর্থ চেষ্টা।

Scoped — worksসীমাবদ্ধ — কাজ করে

"Migrate all fetch() calls in /src/services/userService.ts from APIv1 to APIv2. Replace api.get() with apiClient.get(). Preserve error handling. Return only changed lines."

Each agent needs ALL context in its prompt — agents don't share memory. If agent A discovers something, agent B won't know. প্রতিটা এজেন্টের prompt-এ সব context দাও — এজেন্টরা মেমরি শেয়ার করে না।

WATCH OUT

Subagents have no shared memory. সাবএজেন্টের শেয়ার্ড মেমরি নেই।

Agent 47 discovers
"The legacy auth module uses SHA-1 hashing — everything that touches it needs to handle this." "লেগ্যাসি auth মডিউল SHA-1 হ্যাশিং ব্যবহার করে।"

Agent 23 does not know this. It assumes SHA-256. It ships broken code. Agent 23 এটা জানে না। সে SHA-256 ধরে নেয়। ভাঙা কোড ship করে।

Fix: pass critical discoveries explicitly in every subagent's prompt. Or use Agent Teams where agents can message each other. সমাধান: গুরুত্বপূর্ণ তথ্য প্রতিটা subagent-এর prompt-এ দাও। বা Agent Teams ব্যবহার করো।

WATCH OUT

Rate limits are real. Control concurrency. Rate limit বাস্তব। concurrency নিয়ন্ত্রণ করো। 

# BAD — spawning 200 agents instantly → API throttling
ls src/**/*.ts | xargs -P 200 -I {} claude -p "Migrate {}"

# GOOD — 10 concurrent max, plus max-turns guard
ls src/**/*.ts | xargs -P 10 -I {} \
  claude -p "Migrate {}" --max-turns 15

# GOOD — orchestrator uses Task tool internally (managed concurrency)
claude -p "Migrate all fetch() calls across src/ — spawn agents as needed"
  • ✅Use --max-turns on every subagent — prevents runaway sessions--max-turns সবসময় দাও — বেলাগাম সেশন ঠেকাও
  • ✅Prefer Task tool (managed) over raw shell xargs for large swarmsবড় ঝাঁকের জন্য Task tool পছন্দ করো
  • ⛔Never spawn unlimited parallel agents — rate limits will throttle youঅসীম সমান্তরাল এজেন্ট spawn করো না
