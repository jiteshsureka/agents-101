Claude Code Skills

Pre-built slash-command skills your team can invoke. Each entry covers what it does, when to trigger it, and real enterprise use cases.

Framework

superpowers

Complete agentic software development methodology built on composable skills. Ships 20+ battle-tested skills: TDD, debugging, collaboration, planning. Slash commands include /brainstorm, /write-plan, /execute-plan.

/plugin install superpowers@claude-plugins-official npx skills add obra/superpowers

Install once per team. Establishes a shared methodology so every dev's Claude sessions follow the same structured workflow.

↗ github.com/obra/superpowers
Productivity

caveman

Cuts ~65% of output tokens by switching Claude to terse caveman mode — drop filler, keep substance, use fragments. Supports lite / full / ultra intensity levels. Works as Claude Code plugin, Codex plugin, Cursor rule, and 40+ other agents.

/caveman · /caveman lite|full|ultra npx skills add JuliusBrussee/caveman

Activate on long sessions or when context window is filling fast. 65% fewer output tokens = meaningfully lower cost and faster responses.

↗ github.com/JuliusBrussee/caveman
Code Intelligence

codegraph

Pre-indexed code knowledge graph that auto-syncs on file changes. Exposes symbol relationships, call graphs, and code structure as an MCP server — agents query the graph instead of scanning files. 100% local, zero file contents leave the machine.

Auto-configured MCP server · no slash command needed npx codegraph install

Large codebases where Claude repeatedly scans the same files. Wires into Claude Code, Cursor, Codex, Gemini CLI, and Kiro automatically on first run.

↗ github.com/colbymchenry/codegraph
Code Intelligence

graphify

Turns any folder of code, SQL schemas, docs, papers, or images into a queryable knowledge graph. Supports 20+ languages via tree-sitter AST (Python, TS, Go, Rust, Java, C, C++, Ruby, C# and more). All processing is local — no file contents leave your machine.

/graphify npm install -g @sentropic/graphify && graphify install

Map an unfamiliar codebase in minutes. Also useful for combining app code + database schema + infrastructure into one queryable graph at project start.

↗ github.com/safishamsi/graphify
