Help Us Grow

agents-101 is open source and community-driven. Found a bug, have a skill to share, or want to suggest an episode? Here's how to help.

🐛

Report Bugs & Suggest Features

Open a GitHub Issue for anything that's broken or could be better — broken slides, wrong content, missing features, episode ideas, skill suggestions.

  • Broken slide navigation or layout
  • Incorrect or outdated content
  • New episode topic ideas
  • Feature requests for the site or decks
  • Missing skills or resources
Open an Issue ↗
🚀

Submit a Pull Request

Want to add a skill, vlog, blog post, or resource? Fork the repo and submit a PR. Small, focused PRs merge fastest.

  • Add a skill — add a card to skills/index.html with category, description, invoke command, and GitHub link
  • Add a vlog or blog — add a card to vlogs/index.html
  • Add a resource link — add to resources/index.html
  • Fix a bug — any file, any fix
View Pull Requests ↗
